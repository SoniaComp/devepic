* 소비자
    => 소비자는 회원 가입을 할 수 있다.
    => 소비자는 로그인을 할 수 있다.
    => 소비자는 서비스에서 자신을 탈퇴시킬 수 있다.
    => 소비자는 비밀번호를 찾을 수 있다.

    => 소비자는 자신의 프로필을 설정할 수 있다.
    => 소비자는 자신의 프로필을 수정할 수 있다.
    
    => 소비자는 메뉴를 조회할 수 있다.
    => 소비자는 공급자를 조회할 수 있다.

    => 소비자는 주문을 할 수 있다.
    => 소비자는 주문을 조회할 수 있다.

    => 소비자는 자신의 주문에 대한 리뷰를 남길 수 있다.
    => 소비자는 자신이 남긴 리뷰를 조회할 수 있다.
    => 소비자는 자신이 남긴 리뷰를 수정할 수 있다.
    => 소비자는 자신이 남긴 리뷰를 삭제할 수 있다.


* 공급자
    => 공급자는 로그인을 할 수 있다.
    => 공급자는 비밀번호를 찾을 수 있다.

    => 공급자는 새로운 메뉴를 등록할 수 있다.
    => 공급자는 등록된 메뉴를 조회할 수 있다.

    => 공급자는 프로필을 설정할 수 있다.
    => 공급자는 프로필을 수정할 수 있다.

    => 공급자는 자신에게 들어온 주문을 조회할 수 있다.
    => 공급자는 자신에게 들어온 주문을 수정할 수 있다.

    => 공급자는 자신에게 들어온 주문의 리뷰에 답글을 달 수 있다.
    => 공급자는 자신에게 들어온 주문의 리뷰를 조회할 수 있다.

    => 공급자는 자신에게 들어온 주문들의 통계치를 확인할 수 있다.


* 관리자
    => 관리자는 공급자를 추가할 수 있다.
    => 관리자는 주문 / 메뉴 / 소비자 / 공급자를 조회할 수 있다.
    => 관리자는 주문 / 메뉴 / 소비자 / 공급자를 수정할 수 있다.
    => 관리자는 주문 / 메뉴 / 소비자 / 공급자를 삭제할 수 있다.
