* User
    => 가입
        아이디, 비밀번호, 이메일 입력
        아이디 => 8 자 이상 20 미만
        아이디 => 중복 X
        아이디 => 영어 소문자, 숫자 조합으로만 생성 가능

        비밀번호 => SHA256로 hashing 후 저장
        비밀번호 => 8 자 이상 20 미만

        이메일 => @ 이후 . 필수
        
    => 로그인
        아이디, 비밀번호 검증
        
    => 탈퇴
        탈퇴 사유 기록
        탈퇴 시점 기록
        
    => 비밀번호 찾기
        가입 시 등록한 이메일로 임시 비밀번호 발급

    => 비밀번호 수정
        기존 비밀번호 검증 후 수정 가능
        새로운 비밀번호의 검증 방식은 가입 시와 동일

        프로필 설정과 동일

        Order::가격,

        Order::요구사항,
        Order::주문 상태,
        Order::결제 수단

    => Order::리뷰 생성

        Order::주문,

        Order::리뷰 생성 시각

        Order::주문 상태가 완료로 설정되어야 생성 가능

* Provider
    => 프로필 설정
        이름 => 3자 이상 10자 이하
        소개 =>
            소개글 => 128자 미만
            영업 시간 => 시작시, 종료시
            전화번호 => - 제외 11자 미만
            배달가능지역 => 동네명
            배달팁 => 최소 배달팁 (₩), 최대 배달팁 (₩) 
            최소주문금액 => (₩)

        사업자 정보 =>
            대표자 => 2자 이상 30자 미만
            상호명 => 30자 미만
            사업자주소 => Location::주소
            사업자등록번호 => 12자 (-포함)

        Location::주소

    => 프로필 수정
        Provider::프로필 소개 수정 가능
    => 메뉴 생성
        Provider::메뉴 이름, Provider::설명, Provider::메뉴 가격
    => 메뉴 수정
        메뉴 생성과 동일
    => Order::주문 조회
    => Order::주문 수정
        Order::주문 상태 수정 가능
        Order::주문 수정 시각 기록
    => Order::주문 통계
        메뉴 별 Order::주문 수, Order::주문 가격 합
        기간 별 Order::주문 수, Order::주문 가격 합
    => Order::리뷰 조회
    => Order::리뷰 답글 생성
        자신을 대상으로 한 Order::리뷰에 답글 생성 가능
        답글 => 40자 미만
        Order::리뷰 당 1개로 제한
    => Order::리뷰 답글 수정
        Order::리뷰 답글 생성과 동일
